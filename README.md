# addon-hth-archive

## 项目简介

`addon-hth-archive` 是一个用于归档和发布独立 HTML 页面的仓库。  
本仓库本身不针对任何特定域名或网站，仅作为静态 HTML 文件的存储与分发空间。  
所有页面均为独立、自包含的 HTML 文件，便于直接查看、分享或离线使用。

## 目录结构

```
addon-hth-archive/
├── pages/          # 存放归档的 HTML 页面文件
├── assets/         # 页面可能引用的资源文件（如 CSS、JS、图片等）
├── index.html      # 可选的主页索引文件
└── README.md       # 本文件
```

- **pages/**：主要归档目录，每个 HTML 文件作为一个独立页面存放于此。
- **assets/**：若页面需要外部资源（样式、脚本、媒体等），可统一放置于此，便于管理。
- **index.html**：可选，用于展示页面列表或提供导航。

## 页面归档说明

- 所有归档页面均为独立的 `.html` 文件，可直接在浏览器中打开运行。
- 页面内容不涉及任何特定域名或网站，仅供记录与存档用途。
- 归档的页面可能来自不同的来源或用途，但均经过整理与归类。
- 本仓库不保证归档内容的时效性、准确性或可用性，仅以“原样”提供。

## 维护说明

- 出于简洁考虑，本仓库不包含复杂的构建流程或部署脚本，所有文件均为静态资源。
